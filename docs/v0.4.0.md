# v0.4.0 - AI 宠物自主生成与服务配置重磅发布

## 日期
2026-05-20

## 变更内容

本版本迎来了 AIPet 项目的里程碑升级——**「AI 宠物自主生成系统（AI Pet Creator）」**。用户现在可以仅通过文字描述或上传参考图，利用最先进的 AIGC 图像大模型一键生成完整动作集（Spritesheet 雪碧图）的专属桌面宠物，并无缝导入宠物库进行交互。

---

### 1. AI 宠物生成管线 (AI Pet Creator Pipeline)
- **多阶段自动化管线**：核心后端采用 `tokio` 异步管道，将宠物制作分为「准备工作目录」->「生成基础形象」->「用户预览确认」->「逐行动作序列帧生成」->「条带拼接与合图导出」->「生成 pet.json 与导入库」六大步骤。
- **用户确认阻断器**：首创「基础形象确认机制」，在生成第 1 帧基础图后自动暂停管道，将底图反馈给前端渲染。用户确认画风和角色设计无误后，再推进多达数十帧的九大基础动作条带（Run、Jump 等）批量异步生成。
- **任务断点续跑**：支持临时工作目录（`tmp/`）的进度文件 `progress-creation.json` 自动落盘。若生成在中间步骤被用户主动取消、或因网络/API Key 额度不足而中断，后续可在「未完成任务」列表中一键**断点续跑**，跳过已完成动作，保障额度不浪费。
- **智能雪碧图合成**：利用 Rust 图像处理库 `image` 与 `webp`，接收 AI 接口返回的多动作序列帧拼图，进行高精度的色度抠图（Chroma Key）去绿幕背景，提取帧图案并合并拼接成 `8x9` 或特定列宽的单张无背景 WebP 序列条带图，大幅优化运行时显存开销。

### 2. 多模型 AI 网关与配置 (AI Configuration & Provider Cache)
- **多服务商本地缓存**：新增 `ai-config.json` 独立配置文件。支持配置不同的 AI 服务商（如阿里云 DashScope 万象 Wan2.7 图像大模型等），内置缓存机制（`Provider Cache`），各厂商的 API Key、Endpoint、Model 参数可独立保存并一键切换。
- **配置防丢/安全过滤**：在应用设置页新增「AI 服务配置」卡片，支持密钥输入隐藏、自定义 Base URL 以及测试，后端命令自动完成路径创建与安全持久化。

### 3. AI 提示词构建器 (Prompt Builder)
- **动作形态提示词引擎**：内置精密设计的 AIGC 图像生成提示词模板。能够将用户的宠物名称、设定描述和风格预设（如 3D 粘土、像素风、2D 二次元、像素动画等）智能翻译成适配万象大模型的结构化 Prompt。
- **抠图与动作姿态指令**：在每行 Prompt 后自动追加高度一致的“Chroma key 抠图”绿幕指令，以及高保真的九大状态（待机、向右跑、向左跑、挥手、跳跃、失败、等待、专注、审视）运动姿态描述指令，确保各行动作连贯。

### 4. 全新「AI 宠物制作」前端面板 (Pet Creator UI)
- **高颜值交互向导**：提供完整的「AI 宠物制作」可视化页签，由名称设定、参考图上传、风格预设三大步骤引导。
- **拖拽式上传体验**：精心打磨的参考图拖拽区域，支持上传限制校验，在拖拽文件进入时提供灵动的高亮反馈效果。
- **3D 悬浮预览组件**：生成的 Base Image 支持在 3D 风格卡片中渲染，配合仿物理拖拽摇摆微动效，带来极佳的视觉 QA 交互体验。
- **可视化进度与实时日志**：
  - 各动作（Run、Jump、Idle 等）的生成进度以步骤条与进度环实时展示，精确计算已用时间。
  - 内设「实时终端日志抽屉」，可一键展开查阅后端 Rust 管道正在吐出的详细流式生成日志，方便开发者和高级用户进行链路排查。

## 依赖变更

### Rust
- `uuid` (v1.4+) -- 用于生成高维唯一的 API 任务 ID 与未完任务工作目录名
- `chrono` (v0.4) -- 处理配置项及日志的时间戳格式化
- `image` (v0.25) -- 进行精细色度抠图（绿幕背景去除）、图像分帧裁剪与雪碧图纵向合图操作
- `webp` (v0.3) -- 支持高压缩比、透明通道的 `.webp` 格式雪碧图高效导出

### 前端
- 无新增外部依赖，完全基于 React 18 与 Tailwind CSS v4 原生特性构建
